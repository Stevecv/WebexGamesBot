Getting Started

Bot with a couple of non-time dependant chat based games

Set up for development

  1. Run in IDE - Make sure to replace the auth token with the token from here

.\ngrok.exe config add-authtoken <your auth token>

  1. Then open the port for development

.\ngrok.exe http 12000

  1. Edit the access token in AccessToken and replace it with your own token.

  2. Run the bot

python Main.py
