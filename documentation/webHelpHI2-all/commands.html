Commands

Commands can be run through pinging the bot and entering a string.

Creating a command

  1. Create a new file in commands

  2. Paste

import Utils def test(sender, roomId): print("Send command") Utils.send_message_in_room(roomId, "Send message")

  1. Change name & body of function

  2. Navigate to CommandsMain, registerCommands() and add a line

registerCommand("test", Test.test, "Simple test command")

This will create a new command that can be run by messaging @gamebot@webex.bot test
